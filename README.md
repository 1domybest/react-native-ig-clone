# 기능

- [구글로 로그인]
- [ScrollView 를 이용한 피드목록 UI]
- [마이페이지 UI 구현]
- [마이페이지 상단 오른쪽 버튼 클릭시 바텀시트 활성화 기능]
- [바텀시트 안에 로그아웃기능 구현]
- [새 게시물 만들기 탭이동 기능]
- [사진 업로드및 미리보기 구현]
